# 🍽️ Sangeetha Veg

A modern, performance-focused food ordering website inspired by traditional and contemporary Indian vegetarian cuisine.  
Built with a focus on **UI/UX, performance optimization, and scalable frontend architecture**.

---

##  Live Demo
👉 https://sangeetha-veg.vercel.app/

---

##  Tech Stack

- ⚛️ React (Vite)
- 🎨 Tailwind CSS
- ⚡ Vercel (Deployment)
- 📦 JavaScript (ES6+)
- 🔄 Lazy Loading & Code Splitting
- 🧠 React Hooks (useState, useEffect, useRef)

---

##  Features

-  Fast loading with lazy-loaded components
-  Fully responsive design (mobile-first approach)
-  Smooth scrolling navigation
-  Component-based architecture
-  Optimized rendering using Suspense & lazy loading
-  Clean and modern UI design
-  Modular and reusable components
